3.7.2

use Algorithm 3.22 to simulate the NFA's:

  1. Fig. 3.29.
  2. Fig. 3.30.

on input aabb.

Answer

  1. -start->{0}-a->{0,1}-a->{0,1,2}-b->{0,2,3}-b->{0,2,3}
  2. -start->{0,1,2,3}-a->{0,1,2,3}-a->{0,1,2,3}-b->{0,1,2,3}-b->{0,1,2,3}
